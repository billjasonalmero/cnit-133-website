Homework Part 1: Student's Grade

For Homework 2: Part 1, The project is to built a JavaScript input to check if the student got a grade of A, B, C, D, F. If the Student acquired a grade of 70-100 (C-A) the student passed the course. But if the student acquired a grade of 69 and below (D or F). The student must retake the course It's a "Student Grade" calculator that input's a student's homework average, mid-term exam score, final exam and participation (all those grades will be entered as integers).
Student Grade Formula: final average = (.5 * hwAvg) + (.2 * midExam) + (.2 * finalExam) + (.1 * participation)


Homework Part 2: Sales Commision

For Homework 2: Part 2, The project is to build a JavaScript that calculates the items sold and providing an input of how many of the items was sold inorder to get the "Product" of each item. All four items (item #1, item#2, item#3, item#4) will be added together and 9% of the total items will be added plus an extra $200.00 as the sum of the sold items by the salesperson. jQuery UI tooltip will display a message "Please make sure that hte sales person is spelled correctly".


Homework Part 3: Temperature Calculator

For Homework 3: Part 3, The project is to build a JavaScript that can calculate and convert the given temperatue and from fahrenheit to celcius or celcius to fahrenheit. Using th e formula:
C = 5/9 * (F-32)
F = (9/5*C +32)
There are two buttons "Fahrenheit to Celcius" and "Celsius to Fahrenheit". 0 degrees Celcius = 32 degrees Fahrenheit.
